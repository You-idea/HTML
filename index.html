Mediation

R

这是一个基于matlab、python和R的中介效应分析模型代码库，涵盖了简单中介分析和链式中介分析。

MIT

Gradient

Matlab

这是一个用于连接组皮层梯度计算的matlab代码，广泛应用于脑科学和医学等多个领域。

GPL-3.0

ComBat

Matlab

这是一个基于matlab开发的用于消除数据间批次效应的数据校正方式，能够通过合并小样本增加样本量。

MIT

LOOCV

Python

这是一个基于python的用于三折交叉验证的留一法计算工具，其运算逻辑广泛应用于对模型性能的评估。

MIT

IS-RSA

Python

这是一个基于python的个体间表征相似性（IS-RSA）的计算代码，其中还包括IS-RSA和Mediation的整合代码。

MIT

Behaviour

Python

这是一个基于python的用于大规模被试的批量问卷整理的代码，可以提高我们在数据筛选阶段的工作效率。

MIT
